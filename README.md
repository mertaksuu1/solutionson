## Getting Started

This project is a starting point for a Flutter application. 
First, run your emulator and then run the code.
On this page you can click anywhere, it will be running but all of them useless because We haven't used firebase yet.
In the top left menu, there are achievements, community and about us pages.
On the top right there is the sign up and achievements section.
You can create a campaign when you press the + button at the bottom right.
